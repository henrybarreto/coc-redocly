# coc-redocly

Adpter to VsCode Redocly extensions.

## Install

`:CocInstall coc-redocly`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-redocly-keymap)`

## Lists

`:CocList demo_list`
